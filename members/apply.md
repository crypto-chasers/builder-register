```Java#
public class Main {
    public static void main(String []args) {
        int limit=100;
        double sum = 0.0;
        int n=1;
        do
        {
            sum = sum + (Math.pow(n,3) - Math.pow(n, 1/3));
            n++;
        }
        while(n<=limit);
        System.out.println("sum="+sum); // sum=2.55024E7
    }
}
```
